Detailed Description

Data-parallel versions of core computational functions using OpenMP.

Author
Mark J. Olah (mjo@cs.unm.edu)
Date
2013-2017 OpenMP computation for stacked Model operations on verctor data.
  • Design Decisions
  • Because we want to integrate as seamlessly as possible with matlab, we use the armadillo package which stores arrays in column major order.
  • Therefore in the *_stack operations, if they are to be parallelized, we want the data stored as a nParms X n matrix, i.e. each column is a parameter matrix. Simillarly stacks are size X size X n, so that contiguous images sequences are contiguous in memory. This avoids false sharing.
