Detailed Description

template<class Model>
class mappel::ThreadedEstimator< Model >

We avoid combining Estimator and ThreadedEstimator classes so that a future GPU implementation can inherit directly from Estimator as it will present a differnt method for estimate_stack pure virtual member function. For now all other (CPU) estimators inherit from ThreadedEstimator.
